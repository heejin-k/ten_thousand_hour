## 배운 점 & 문제점

- jQury를 아직 안배웠지만, 검색해보며 모달창 띄우는 정도는 구현을 해보며 jQury와 친숙해진 것 같다.
- 반응형에서 라이캣 이미지 여부를 가상요소를 사용하여 컨트롤하였으며, 모달창의 버튼 사이즈가 달라지면서 2줄로 출력될 경우 " 훈련하러 가기" 부분을 내리기위해 여러 방법을 시도해봤으나 결국 span태그를 이용하여 display:block으로 해결했다.
